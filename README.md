# SudokuX — Premium Sudoku Puzzle Generator

![SudokuX Banner](https://images.unsplash.com/photo-1528819622765-d6bcf132f793?q=80&w=2070&auto=format&fit=crop)

SudokuX is a high-performance, modern Sudoku application built with pure Vanilla JavaScript, HTML5, and CSS3. It features a stunning glassmorphism design, real-time puzzle generation, and a suite of premium features designed for the ultimate Sudoku experience.

## ✨ Features

- **🎯 Advanced Puzzle Engine**: Dynamically generates unique puzzles with three difficulty levels: **Easy**, **Medium**, and **Hard**.
- **🎨 Glassmorphism UI**: A beautiful, modern interface with smooth gradients, animated background orbs, and dark/light mode support.
- **📊 Real-time Stats**: Track your progress with a built-in timer, mistake counter, and hint tracking.
- **💡 Smart Hints**: Get help when you're stuck without giving away the whole game.
- **🔄 State Persistence**: Your progress is automatically saved to local storage, allowing you to resume your game anytime.
- **🎉 Win Celebrations**: Interactive confetti animations and detailed performance modals when you solve a puzzle.
- **🔊 Interactive Sound Effects**: Atmospheric sound feedback for interactions (can be toggled).
- **📱 Fully Responsive**: Optimized for both desktop and mobile devices.

## 🚀 Tech Stack

- **HTML5**: Semantic structure and ARIA labels for accessibility.
- **CSS3**: Custom properties, Glassmorphism, CSS Animations, and Flexbox/Grid.
- **JavaScript (ES6+)**: Core logic, puzzle generation algorithms, and DOM manipulation.

## 🎮 How to Play

1. Select your preferred **Difficulty** level.
2. Click on any cell in the 9x9 grid to select it.
3. Use the **Number Pad** or your keyboard keys (1-9) to enter values.
4. Use the **Hint** button if you need a little help.
5. Solve the puzzle correctly to see the win celebration!
